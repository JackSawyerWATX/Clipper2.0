# Clipper 2.0

A React application with 10 pages built using Vite and React Router.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Structure

- `src/pages/` - Contains all 10 page components (Page1.jsx through Page10.jsx)
- `src/App.jsx` - Main app component with routing configuration
- `src/main.jsx` - Application entry point

## Pages

The application includes 10 pages accessible via the navigation bar:
- Page 1 (Home) - `/`
- Page 2 - `/page2`
- Page 3 - `/page3`
- Page 4 - `/page4`
- Page 5 - `/page5`
- Page 6 - `/page6`
- Page 7 - `/page7`
- Page 8 - `/page8`
- Page 9 - `/page9`
- Page 10 - `/page10`

Each page is a separate React component ready for customization.
